# OsakaJade

A serene sage green and warm beige Spotify theme inspired by the Omarchy "Osaka Jade" desktop environment theme. This theme brings a cohesive, nature-inspired aesthetic to your Spotify experience.

## Screenshots

![OsakaJade Preview](screenshot.png)

## Features

- **Harmonious Color Palette**: Soft sage greens, warm beiges, and subtle cyan accents
- **Calming Aesthetic**: Nature-inspired colors designed to reduce visual fatigue
- **Modern Interactions**: Smooth transitions and hover effects for an engaging UI
- **Consistent Design**: Colors work across all Spotify UI elements

## Color Scheme

| Element | Color | Hex |
|---------|-------|-----|

| Buttons/Active Elements | Cyan | #2DD5B7 |
| Highlights | Magenta | #D2689C |
| Accents | Forest Green | #549e6a |
| Errors | Red | #FF5345 |
